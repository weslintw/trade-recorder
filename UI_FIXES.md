### 1. 資訊排列優化
**問題**：進場價格、平倉價格、手數、時間分散顯示

**解決方案**：
- 將品種、方向、盈虧放在第一行（標題行）
- 將價格、手數、時間整合在第二行（摘要行）
- 使用淺灰色背景區分
- 資訊更緊湊，一目了然

```

│ XAUUSD  📈 做多            +2.00    │ ← 標題行

│ #標籤 #標籤                         │
│ 筆記內容...                         │

└─────────────────────────────────────┘
```

## 📝 如果圖片還是看不到

請檢查：

1. **MinIO 是否運行**：
   ```powershell
   # 檢查 9000 端口
   netstat -ano | findstr :9000
   ```

2. **後端是否運行**：
   ```powershell
   # 檢查 8080 端口
   netstat -ano | findstr :8080
   ```

3. **瀏覽器 Console**：
   - 按 F12 開啟開發者工具
   - 查看 Console 是否有錯誤
   - 查看 Network 標籤，檢查圖片請求狀態

4. **測試圖片 URL**：
   - 複製圖片 URL
   - 直接在瀏覽器開啟
   - 看是否能顯示
